\DoxyCodeLine{36 \textcolor{comment}{   *}}

\DoxyCodeLine{41 \textcolor{comment}{   *   LCD Filtering}}
\DoxyCodeLine{42 \textcolor{comment}{   *}}
\DoxyCodeLine{43 \textcolor{comment}{   * @abstract:}}
\DoxyCodeLine{44 \textcolor{comment}{   *   Reduce color fringes of LCD-\/optimized bitmaps.}}
\DoxyCodeLine{45 \textcolor{comment}{   *}}
\DoxyCodeLine{46 \textcolor{comment}{   * @description:}}
\DoxyCodeLine{47 \textcolor{comment}{   *   The @FT\_Library\_SetLcdFilter API can be used to specify a low-\/pass}}
\DoxyCodeLine{48 \textcolor{comment}{   *   filter, which is then applied to LCD-\/optimized bitmaps generated}}
\DoxyCodeLine{49 \textcolor{comment}{   *   through @FT\_Render\_Glyph.  This is useful to reduce color fringes}}
\DoxyCodeLine{50 \textcolor{comment}{   *   that would occur with unfiltered rendering.}}
\DoxyCodeLine{51 \textcolor{comment}{   *}}
\DoxyCodeLine{52 \textcolor{comment}{   *   Note that no filter is active by default, and that this function is}}
\DoxyCodeLine{53 \textcolor{comment}{   *   *not* implemented in default builds of the library.  You need to}}
\DoxyCodeLine{54 \textcolor{comment}{   *   \#define FT\_CONFIG\_OPTION\_SUBPIXEL\_RENDERING in your `ftoption.h' file}}
\DoxyCodeLine{55 \textcolor{comment}{   *   in order to activate it.}}
\DoxyCodeLine{56 \textcolor{comment}{   *}}
\DoxyCodeLine{57 \textcolor{comment}{   *   FreeType generates alpha coverage maps, which are linear by nature.}}
\DoxyCodeLine{58 \textcolor{comment}{   *   For instance, the value 0x80 in bitmap representation means that}}
\DoxyCodeLine{59 \textcolor{comment}{   *   (within numerical precision) 0x80/0xFF fraction of that pixel is}}
\DoxyCodeLine{60 \textcolor{comment}{   *   covered by the glyph's outline.  The blending function for placing}}
\DoxyCodeLine{61 \textcolor{comment}{   *   text over a background is}}
\DoxyCodeLine{62 \textcolor{comment}{   *}}
\DoxyCodeLine{63 \textcolor{comment}{   *   \{}}
\DoxyCodeLine{64 \textcolor{comment}{   *     dst = alpha * src + (1 -\/ alpha) * dst    ,}}
\DoxyCodeLine{65 \textcolor{comment}{   *   \}}}
\DoxyCodeLine{66 \textcolor{comment}{   *}}
\DoxyCodeLine{67 \textcolor{comment}{   *   which is known as OVER.  However, when calculating the output of the}}
\DoxyCodeLine{68 \textcolor{comment}{   *   OVER operator, the source colors should first be transformed to a}}
\DoxyCodeLine{69 \textcolor{comment}{   *   linear color space, then alpha blended in that space, and transformed}}
\DoxyCodeLine{70 \textcolor{comment}{   *   back to the output color space.}}
\DoxyCodeLine{71 \textcolor{comment}{   *}}
\DoxyCodeLine{72 \textcolor{comment}{   *   When linear light blending is used, the default FIR5 filtering}}
\DoxyCodeLine{73 \textcolor{comment}{   *   weights (as given by FT\_LCD\_FILTER\_DEFAULT) are no longer optimal, as}}
\DoxyCodeLine{74 \textcolor{comment}{   *   they have been designed for black on white rendering while lacking}}
\DoxyCodeLine{75 \textcolor{comment}{   *   gamma correction.  To preserve color neutrality, weights for a FIR5}}
\DoxyCodeLine{76 \textcolor{comment}{   *   filter should be chosen according to two free parameters `a' and `c',}}
\DoxyCodeLine{77 \textcolor{comment}{   *   and the FIR weights should be}}
\DoxyCodeLine{78 \textcolor{comment}{   *}}
\DoxyCodeLine{79 \textcolor{comment}{   *   \{}}
\DoxyCodeLine{80 \textcolor{comment}{   *     [a -\/ c, a + c, 2 * a, a + c, a -\/ c]    .}}
\DoxyCodeLine{81 \textcolor{comment}{   *   \}}}
\DoxyCodeLine{82 \textcolor{comment}{   *}}
\DoxyCodeLine{83 \textcolor{comment}{   *   This formula generates equal weights for all the color primaries}}
\DoxyCodeLine{84 \textcolor{comment}{   *   across the filter kernel, which makes it colorless.  One suggested}}
\DoxyCodeLine{85 \textcolor{comment}{   *   set of weights is}}
\DoxyCodeLine{86 \textcolor{comment}{   *}}
\DoxyCodeLine{87 \textcolor{comment}{   *   \{}}
\DoxyCodeLine{88 \textcolor{comment}{   *     [0x10, 0x50, 0x60, 0x50, 0x10]    ,}}
\DoxyCodeLine{89 \textcolor{comment}{   *   \}}}
\DoxyCodeLine{90 \textcolor{comment}{   *}}
\DoxyCodeLine{91 \textcolor{comment}{   *   where `a' has value 0x30 and `b' value 0x20.  The weights in filter}}
\DoxyCodeLine{92 \textcolor{comment}{   *   may have a sum larger than 0x100, which increases coloration slightly}}
\DoxyCodeLine{93 \textcolor{comment}{   *   but also improves contrast.}}
\DoxyCodeLine{94 \textcolor{comment}{   */}}
